# SoliCurses-C
A remake of SoliCurses in C with improved UI

This game is compatible with *nix OSes and Windows (with Cygwin)

Dependencies:

    *curses
    
    a compiler
    
    a terminal environment with color
   
How to build and run:
    
    Unix:
    
        make

        // Or

        make run
